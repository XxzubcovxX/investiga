# 🕵️‍♂️ Investiga

O **Investiga** é um WebApp criado para facilitar a consulta e a transparência de dados sobre figuras públicas. O objetivo é centralizar informações de forma acessível para que qualquer cidadão possa pesquisar e entender melhor o perfil de seus representantes.

---

## 🛠️ Tecnologias Utilizadas (Stack)

O projeto foi construído utilizando o que há de mais moderno para garantir velocidade e organização:

| Camada | Tecnologia | Descrição |
|--------|------------|-----------|
| **Frontend** | React.js | Interface dinâmica e responsiva |
| **Backend** | FastAPI | API em Python de alta performance |
| **Banco de Dados** | MySQL | Armazenamento robusto de dados |
| **Infraestrutura** | Docker & Docker Compose | Padronização do ambiente |
| **Automação** | GitHub Actions | Fluxo de deploy automatizado |

---

## 🎯 Funcionalidades Principais

- Busca rápida de políticos.
- Listagem detalhada de candidatos.
- Integração com inteligência artificial e machine learning para análise de dados *(em desenvolvimento)*.
